A modern, Flask-based Human Resource Management System designed for small to medium-sized businesses. This platform provides distinct interfaces for Company Owners, HR Teams, and Employees to manage daily operations efficiently.

## Features

* **Role-Based Access Control**: Customized dashboards for Owners, HR, Managers, and Employees.
* **Time & Attendance**: Real-time "Clock In/Out" functionality with automated attendance logging.
* **Leave Management**: Employees can request leave; Managers/HR can approve or reject requests via their dashboard.
* **Professional Profiles**: User-friendly profile management including basic info updates and secure password changes.
* **Work in Progress**: 
    * [ ] Payroll & Payslip Generation
    * [ ] Advanced Reporting & Analytics
    * [ ] Document Management
